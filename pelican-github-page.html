Pelican发布到github page

Date Fri 21 March 2014 Tags pelican github

在github上创建和用户名同名的工程

在github上面创建和用户名相同的工程。如我的用户名是mingoc， 可以创建mingoc工程，上传到mingoc的master分支的文件将自动发布。可以通过 mingoc.github.com 或者 mingoc.github.io 访问。

发布output的内容到github

Pelican 使用make html 生成的html文件在output文件夹里面。git push -f 使用force方式push到github

cd $pelican_blog_dir
make html 
cd output
git init 
git add .
git commit -am 'add more content'
git remote add origin git@github.com:mingoc/mingoc.github.com.git   
git push -f origin master
