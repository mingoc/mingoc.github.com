取消Pelican代码block中异常的红色方块

Date Fri 21 March 2014 Tags pelican pygment

Pelican生成的代码block中有一些& $ 等字符高亮显示为红色。看着不正常。可通过修改pygments的css文件来取消。

首先确定你的pelican使用的theme

 $grep -i theme pelicanconf.py
 THEME = "/home/mingo/blog/mingoc.github.com/themes/pelican-bootstrap3"

根据你的theme,打开下面这个./themes/pelican-bootstrap3/static/css/pygments/native.css 删除下面这一行

.highlight pre .err { color: #a61717; background-color: #e3d2d2 } /* Error */
